# Sistema Inteligente de Agendamento de Consultas

## Contexto do Projeto

Projeto desenvolvido como solução para o desafio proposto pelo:

**SENAI Vitória da Conquista (BA)**

(https://plataforma.gpinovacao.senai.br/plataforma/demandas-da-industria/interna/11929)

## Sobre o Projeto
Este projeto tem como objetivo desenvolver uma plataforma digital para otimizar o processo de marcação de consultas médicas, tornando-o mais eficiente, acessível e padronizado.

Atualmente, o agendamento é realizado de forma híbrida (presencial, telefone, WhatsApp e e-mail), envolvendo processos manuais que geram atrasos, falhas de comunicação e retrabalho. A solução proposta centraliza e automatiza esse fluxo, melhorando a experiência tanto para pacientes quanto para a equipe administrativa.

---

## Objetivos

- Reduzir o tempo entre solicitação e confirmação de consultas
- Melhorar a experiência do paciente
- Garantir acessibilidade para diferentes perfis de usuários
- Diminuir o absenteísmo com lembretes automáticos
- Centralizar informações e histórico dos pacientes
- Padronizar o processo de agendamento
- Reduzir a sobrecarga da equipe administrativa

---

## Funcionalidades

### Para Pacientes
- Agendamento de consultas online
- Consulta de horários disponíveis em tempo real
- Confirmação de agendamento
- Recebimento de lembretes automáticos (WhatsApp/SMS)
- Interface simples e acessível (foco em idosos e usuários com baixa familiaridade digital)

---

### Para Administração
- Painel de controle de agendamentos
- Visualização de agendas médicas
- Cadastro e gestão de pacientes
- Acesso ao histórico de consultas
- Edição e cancelamento de consultas
- Padronização do fluxo de atendimento

---

### 🤖 Funcionalidades Inteligentes
- Lembretes automáticos de consultas
- Confirmação ativa de presença
- Liberação automática de vagas não confirmadas
- Lista de espera inteligente

##  Fluxo do Sistema

1. Paciente solicita agendamento (site, WhatsApp ou presencial)
2. Sistema valida os dados automaticamente
3. Exibe horários disponíveis
4. Paciente seleciona a opção desejada
5. Confirmação automática da consulta
6. Envio de lembretes antes do atendimento
